# Remanga Downloader на Dart

Этот скрипт позволяет скачивать страницы манги с сайта Remanga, используя идентификатор главы.

## Требования
- Dart SDK (уже установлен в этой среде)
- Пакет `http` (уже добавлен в проект)

## Как использовать

1. Перейдите в директорию проекта:
   ```bash
   cd remanga_project
   ```

2. Запустите скрипт, передав ID главы (число из ссылки):
   ```bash
   dart bin/main.dart 556648
   ```

## Особенности и ограничения
- **ID главы**: ID главы можно найти в URL. Например, в `https://remanga.org/manga/.../556648`, ID — это `556648`.
- **Ошибка 403 (Forbidden)**: Remanga ограничивает доступ к своему API для регионов за пределами СНГ. Если вы запускаете скрипт не из России или стран СНГ, сервер может вернуть ошибку 403. В этом случае рекомендуется использовать VPN с российским IP-адресом.
- **Локальное сохранение**: Страницы сохраняются в папку `downloads/<название_манги>/chapter_<номер_главы>/`.

## Код скрипта
Основной код находится в файле `bin/main.dart`. Вы можете модифицировать его для добавления авторизации через куки, если глава доступна только зарегистрированным пользователям.
